Game Overview

Rotterdam Metro Simulator (RMS for short) is a railway game on Roblox that prioritises the user experience of what Rotterdam and her metrolines were like throughout the years. This game is currently in development, thus there is currently no playability.

Features

  • Various metro types ranging from realistic to unrealistic
  • Stations represented in each era
  • Routes represented in each era
  • Custom consist creation
  • (Almost) every used livery
